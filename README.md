# Restaurant Page 🍽️

A browser-based restaurant website built with vanilla JavaScript and webpack.

## 🎮 Live Demo

[View it here](https://paulperrin-stack.github.io/restaurant-page/)

## Features

- Three-page site: Home, Menu, and About
- Tab navigation with dynamic content rendering — no page reloads
- Menu grid with dish images, names, and descriptions
- About page with opening hours, location, and reservations
- Responsive card layouts built with CSS Grid

## What I Learned

- Setting up and configuring webpack from scratch
- Using ES6 `import`/`export` to split code across modules
- Importing CSS and images directly into JavaScript modules via webpack
- Dynamically creating and injecting DOM elements instead of writing static HTML
- Using CSS custom properties (`--burgundy`, `--gold`, etc.) for consistent theming across files
